Security & Trust

Security is our priority. Here is how Spriters Bot keeps your server and community safe.

Restricted Permissions

The bot only requests necessary permissions. Dangerous administrative actions are restricted to designated server roles.

Scoped Commands

Commands are restricted to specific channels configured by admins, preventing spam in general chats.

2. Anti-Nuke Measures

We act to protect your server from malicious attacks.

  • Rate Limiting: Prevents users from spamming commands to lag the bot.
  • Audit Logs: Moderation actions are logged for transparency.
  • Authorized Roles Only: Sensitive settings can only be changed by users with Administrator permission.
