Privacy Policy

We value your privacy. This document outlines what data Spriters Bot collects, why we collect it, and how we handle it.

1. Data Collection

  • User IDs & Server IDs: We store unique IDs provided by Discord to link settings (like language preferences) to specific users and servers.
  • Message Content (Ephemeral): For features like "Anti-Spam" or "Moderation", the bot processes message content in real-time RAM. We do NOT persist message logs in a database unless explicitly requested via the `/logs` command for moderation audit trails (which are viewable by server admins).
  • Command Usage: We may log command usage statistics to improve performance and debug errors.

2. Why We Use Data

Functionality

To provide core features like custom dates, birthdays, and moderation settings.

Security

To prevent abuse, spam, and malicious use of the bot's features.

3. Data Retention & Rights

Data is retained only as long as the bot is in your server. If you kick the bot, configuration data may be deleted after a grace period.

You have the right to request deletion of your data. Contact the developer via the Support Server.
